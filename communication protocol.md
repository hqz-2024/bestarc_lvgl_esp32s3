# 东莞宝宇森 BTC500DP PRO 设备端通讯协议

---

## 一、数据包结构

每次通讯发送 1 包数据，每包固定 **12 字节**：

| 字段 | 包头 | 设备类型 | 命令码 | 数据区 | 校验码 | 包尾 |
|------|------|---------|-------|-------|-------|------|
| 长度 | 2字节 | 2字节 | 2字节 | 2字节 | 2字节 | 2字节 |
| 示例 | `55 AA` | `00 01` | `XX XX` | `XX XX` | `NN NN` | `55 BB` |

**串口参数：波特率 19200，8N0**

### 1. 包头
固定 `0x55AA`

### 2. 设备类型

**上位机发送时：**
- `0x8000`：APP 已连接
- `0x0000`：APP 未连接

**设备上报时：**
低字节固定标明设备机型，`0x0002` = BTC500DP PRO，后续机型依次递增（`0x0003` 等）。设备上报只能填写自身机型值。

### 3. 命令码
2字节枚举型。`0x0000` = 未知指令，其余值含义见命令码表。

### 4. 数据区
2字节短整数或枚举型，其值含义见命令码表。

### 5. 校验码
```
校验码 = 命令码 + 数据区
```

### 6. 包尾
固定 `0x55BB`

## 二、命令码表

### 设定类（上位机 → 设备）

| 上位机命令码 | 说明 | 设备响应码 | 说明 | 数据区取值 |
|------------|------|----------|------|-----------|
| `0x0200` | 设定钢板/网格/除锈 | `0x8200` | 确认 | `0x0000`(钢板) / `0x0001`(网格) / `0x0002`(除锈) |
| `0x0300` | 设定 2T/4T 功能 | `0x8300` | 确认 | `0x0000`(2T) / `0x0001`(4T) |
| `0x0400` | 设定电流 | `0x8400` | 确认 | 见下方电流范围表 |
| `0x0500` | 设定后气时间 | `0x8500` | 确认 | `0x0003`~`0x000F`（3~15s） |
| `0x0600` | 设定维弧时间 | `0x8600` | 确认 | `0x0003`~`0x000F`（3~15s） |
| `0x0700` | 设定气压单位 | `0x8700` | 确认 | `0x0000`(PSI) / `0x0001`(MPa) / `0x0002`(BAR) |

| 上位机命令码 | 说明 | 设备响应码 | 说明 | 数据区取值 |

| `0x0003` | 查询 2T/4T 功能 | `0x0083` | 上报 | `0x0000`(2T) / `0x0001`(4T) |

| `0x0005` | 查询后气时间 | `0x0085` | 上报后气时间（s） | `0x0003`~`0x000F`（3~15s） |
| `0x0006` | 查询维弧时间 | `0x0086` | 上报维弧时间（s） | `0x0003`~`0x000F`（3~15s） |

## 三、通讯协议细则

### 3.1 错误信息数据包（`0x8100`）

当设备收到控制命令后无法正确执行时，返回此包。数据区内容为错误码。

| 字段 | 包头 | 设备类型 | 命令码 | 数据区（错误码） | 校验码 | 包尾 |
|------|------|---------|-------|--------------|-------|------|
| 长度 | 2字节 | 2字节 | 2字节 | 2字节 | 2字节 | 2字节 |
| 内容 | `55 AA` | `NN NN` | `81 00` | `NN NN` | `NN NN` | `55 BB` |

**错误码表：**

| 错误码 | 说明 |
|--------|------|
| `0x0001` | 设备正在工作，暂不接受 APP 调动显示面板及功能 |

### 3.2 查询流程

上位机每轮依次发送查询命令（`0x0002`~`0x0009`），设备逐一响应上报当前状态。

## 四、蓝牙广播

广播包总长度 **29 字节**，由以下 3 个 AD Structure 组成：

### AD1：蓝牙类型（3 字节）

```
02 01 06
```

| 字节 | 含义 |
|------|------|
| `02` | Length |
| `01` | Type = Flags |
| `06` | Value（LE General Discoverable + BR/EDR Not Supported） |

### AD2：蓝牙名称（5 字节）

```
04 09 'B' 'Y' 'S'
```

| 字节 | 含义 |
|------|------|
| `04` | Length |
| `09` | Type = Complete Local Name |
| `42 59 53` | ASCII "BYS" |

### AD3：自定义数据包（22 字节）

```
16 FF [MAC×6] [设备类型×2] [钢板网格×2] [2T4T×2] [电流×2] [后气×2] [维弧×2] [P/M×2]
```

| 字节偏移 | 长度 | 字段 | 说明 |
|---------|------|------|------|
| 0 | 1 | 包长 | `0x16`（22，不含本字节） |
| 1 | 1 | 元素类型 | `0xFF`（Manufacturer Specific） |
| 2~7 | 6 | 蓝牙 MAC | 设备 MAC 地址 |
| 8~9 | 2 | 设备类型 | `0x0001` = BTC500DP PRO |
| 10~11 | 2 | 钢板/网格 | `0x0000`(钢板) / `0x0001`(网格) / `0x0002`(除锈) |
| 12~13 | 2 | 2T/4T | `0x0000`(2T) / `0x0001`(4T) |
| 14~15 | 2 | 电流（A） | 当前电流值 |
| 16~17 | 2 | 后气时间（s） | 3~15 |
| 18~19 | 2 | 维弧时间（s） | 3~15 |
| 20~21 | 2 | 气压单位 | `0x0000`(PSI) / `0x0001`(MPa) / `0x0002`(BAR) |
