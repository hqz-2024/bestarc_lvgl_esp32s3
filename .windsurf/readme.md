# skills使用
将./skills放到./.augment里面即可


## open-design-skills

将open-design-skills/skills里面的内容放到.augment/skills/里面即可；
将open-design-skills/craft里面的内容放到.augment/rules/里面；
open-design-skills/design-systems里面是一些网站案例配色风格，需要参考可以将里面的对应风格的md文件粘贴本地并调用
